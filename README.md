# RPG_Python
Projeto de um sistema de jogo RPG inspirado em Elden Ring e implementado em Python

# ⚔️ Mini RPG em Python

Um mini RPG desenvolvido em Python no terminal, focado em praticar lógica de programação, estruturas de dados e funções. O jogador pode criar um personagem, escolher uma classe, enfrentar inimigos e evoluir ao longo das batalhas.

## 🚀 Funcionalidades

* Criação de personagem
* Sistema de classes
* Sistema de batalha
* Inventário
* Ganho de runas
* Progressão de atributos
* Combate em turnos

## 🧠 Conceitos utilizados

* Condicionais (`if/else`)
* Laços de repetição (`while` e `for`)
* Funções
* Listas
* Dicionários
* Biblioteca `random`
* Biblioteca `time`

## 🎮 Tecnologias

* Python 3

## 📚 Objetivo do projeto

Este projeto foi criado com o objetivo de praticar fundamentos de programação em Python através do desenvolvimento de um sistema de RPG simples no terminal.

## ▶️ Como executar

```bash
python RPG.py
```
